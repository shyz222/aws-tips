# AWS-Certified-SOA
## 目的
* AWSのOperation/Security/Governance関連の知識習得
## 背景
* 業務では付加価値に直結するサービス開発にばかり従事しているため習得機会がない
* 一方で、長期的なサービス稼働・運用を考慮するとOperation/Security/Governanceは運用コストに大きく関わってくるためベスプラを学んでおきたい
